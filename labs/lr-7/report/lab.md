# Лабораторная работа 7. Многослойный персептрон (MLP)

## 1. Цель
Классификация грибов (съедобные/ядовитые) с помощью нейронной сети.

## 2. Датасет
- Источник: UCI Mushroom (id=73)
- 8124 образца, 22 категориальных признака
- ~117 признаков после one-hot encoding
- 2 класса: edible / poisonous (~52% / ~48%)

## 3. Предобработка
- Пропущенные значения в stalk-root ('?'): обработаны как отдельная категория
- One-hot encoding всех 22 категориальных признаков
- Стратифицированное разбиение 80/20

## 4. Архитектура MLP
- Вход: ~117 нейронов
- Скрытый слой 1: 64 нейрона, сигмоида
- Скрытый слой 2: 32 нейрона, сигмоида
- Выход: 1 нейрон, сигмоида (P(poisonous))

### 4.1. Обучение
- Функция потерь: binary cross-entropy

- Инициализация весов: Xavier/Glorot
