# Лабораторная работа 7. MLP

## Цель

Создать архитектуру многослойного персептрона, классифицирующего грибы.

## Датасет

[Mushroom](https://archive.ics.uci.edu/ml/datasets/Mushroom)

## Задание

1. 2 скрытых слоя, функция активации — **сигмоида**, градиентный спуск без оптимизаторов.
2. Выходных нейронов: 1 или 2 по количеству классов (на усмотрение студента).
3. Обучить нейронную сеть.
4. Посчитать **accuracy**, **precision**, **recall**, **F1-меру**.
5. Вывести **ROC-кривую**.
