# Лабораторная работа 1. Работа с данными

## Цель

Провести очистку набора данных и отобрать признаки.

## Датасет

**ID_data_mass_18122012** (лист VU при использовании xlsx)

**Целевые переменные:** `G_total`, `КГФ`

## Задание

1. Удалить строки, где обе целевых переменные пустые.
2. Пропуски заполнить `NaN`.
3. Сохранить названия признаков.
4. Построить гистограмму важности признаков по критерию **gain ratio** (нормированный прирост информации).
5. Построить тепловую карту признаков (матрица корреляций всех признаков со всеми, включая целевые переменные); коэффициент корреляции — по модулю.
6. Для каждого признака построить график распределения с линиями Q1 и Q3.
7. Для каждого признака посчитать элементы описательной статистики.
8. Составить список удаляемых и оставляемых признаков с обоснованием.

## Примечания

- Борьба с выбросами и заполнение пропусков — на усмотрение студента, но должна быть обоснована и проводится **после** расчёта корреляций и важности.
- Нормализация в данной лабораторной **не предполагается**.
