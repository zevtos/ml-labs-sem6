# Лабораторная работа 4. Машины опорных векторов

## Цель

Изучить алгоритмы на основе машин опорных векторов в задаче определения потенциала P300.

## Датасет

[MOABB](https://gitlab.com/impulse-neiry/posts/-/blob/master/post01_simple_p300/post01ru_simple_p300.ipynb)

## Задание

1. Проанализировать набор данных, придумать и обосновать детекцию признаков.
2. Обучить машину опорных векторов определять потенциал P300 (выбор типа машины и функции ядра обосновать).
3. Посчитать **accuracy**, **precision**, **recall**, **F1-меру**.
4. Вывести **ROC-кривую**.
